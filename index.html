    Tipos de ameaças
    Cibersegurança é a prática que protege computadores e servidores,
    dispositivos móveis, sistemas eletrônicos, redes e dados contra ataques maliciosos.

    O termo é aplicável a uma variedade de contextos, desde negócios até computação móvel,
    e pode ser dividido em algumas categorias comuns.
DDoS
(Denial of Service Attack)

Um ataque de negação de serviço inunda sistemas, servidores ou redes com tráfego esgotando assim recursos e largura de banda. Como resultado, o sistema não consegue atender a solicitações legítimas.

Phishing

É uma técnica para tentar adquirir dados confidenciais, como números de contas bancárias, por meio de uma solicitação fraudulenta por email ou websites, na qual o agressor se faz passar por uma empresa legítima ou uma pessoa de confiança.

SQL Injection

SQL Injection é um tipo de ataque informático usado para assumir o controle dos dados de uma base de dados ou roubá-los.

TIPOS DE ATAQUES INFORMÁTICOS

1Denial of Service Attack(DDoS)
Um ataque de negação de serviço inunda sistemas, servidores ou redes com tráfego esgotando assim recursos e largura de banda. Como resultado, o sistema não consegue atender a solicitações legítimas.

Isso é conhecido como ataque distribuído de negação de serviço (DDoS).
2Phishing
É uma técnica para tentar adquirir dados confidenciais, como números de contas bancárias, por meio de uma solicitação fraudulenta por email ou websites, na qual o agressor se faz passar por uma empresa legítima ou uma pessoa de confiança.

É resumidamente, o ato de fazer alguém clicar em um link que permite que um ator mal-intencionado obtenha acesso a informações pessoais ou descarregue malware no dispositivo de uma vítima.
3SQL Injection
É um tipo de ataque informático usado para assumir o controlo dos dados de uma base de dados ou roubá-los. Criminosos virtuais exploram vulnerabilidades em aplicações controladas por dados para inserir código malicioso numa base de dados através de uma instrução de SQL mal-intencionada.

Com isso, eles têm acesso às informações confidenciais contidas no banco de dados.
4Ransomeware
Ransomware é um malware que emprega criptografia para manter as informações da vítima como resgate. Os dados críticos de um usuário ou organização são criptografados para que não possam acessar arquivos bases de dados ou aplicativos.

Um resgate é então exigido para fornecer acesso. O ransomware geralmente é projetado para se espalhar por uma rede e atingir bases de dados e servidores de arquivos e, portanto, pode paralisar rapidamente uma organização inteira.

É uma ameaça crescente, gerando biliões de dólares em pagamentos para cibercriminosos e causando danos e despesas significativas para empresas e organizações governamentais.
5Trojan
Um Trojan consistem num tipo de malware que se disfarça de código ou software legítimo. Uma vez dentro da rede, os hackers podem realizar qualquer ação que um utilizador legítimo possa executar, como exportar arquivos, modificar dados, excluir arquivos ou alterar o conteúdo do dispositivo.

Os Trojans podem ser empacotados em downloads de jogos, ferramentas, aplicações ou até mesmo patches de software. Muitos ataques de Trojan também utilizam táticas de engenharia social, bem como spoofing e phishing, para solicitar a ação desejada no utilizador.

Dicas para se proteger contra ataques informáticos

• Atualize o software e sistema operacional:

Isso significa ter os últimos patches de segurança.

• Não clique em links desconhecidos de sites não seguros:

Esta é uma forma comum de propagação de malware.

• Use palavras-passe fortes:

Garanta que suas palavras-passe não sejam fáceis de descobrir.

• Não abra anexos de e-mail de remetentes desconhecidos:

Estes podem estar infectados por malware.

• Use um software antivírus:

O antivírus detetará e removeráameaças.

• Evite usar redes Wi-Fi não seguras em locais públicos:

Redes não seguras deixam o utilizador vulnerável a ataques "man-in-the-middle".
